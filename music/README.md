# Música ambiente do Playbox

Coloca arquivos `.mp3`, `.ogg`, `.wav`, `.m4a` ou `.flac` aqui. O Playbox toca em ordem embaralhada com fade-in/out, pula automaticamente pra próxima quando termina, e pausa enquanto algum jogo tá rodando.

Arquivos novos só aparecem depois de re-buildar o instalador.

Toggle/volume/skip ficam em **Configurações → Música ambiente**.
