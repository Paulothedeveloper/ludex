# Contribuindo com o Ludex · Contributing

> 🇧🇷 Português abaixo · 🇬🇧 English below

## 🇧🇷 Português

O Ludex é um app **pago** desenvolvido de forma aberta — o código é MIT, e a comunidade ajuda a deixá-lo melhor. (A interface em inglês existe porque um usuário pediu! 💜)

**Como ajudar:**
- 🐛 **Bug?** Abra uma [issue](https://github.com/Paulothedeveloper/ludex/issues/new/choose) com: o que aconteceu, o que esperava, passos pra reproduzir, seu SO (Windows/Android) e a versão do Ludex (Ajustes → Sobre).
- 💡 **Ideia/feature?** Abra uma issue descrevendo a dor que resolve.
- 🌍 **Tradução?** O dicionário está em [`src/ludexI18n.js`](src/ludexI18n.js) (6 idiomas). PRs de correção são bem-vindos.
- ⭐ **Gostou?** Deixe uma estrela e [compre o Ludex](https://pauloadriel98.gumroad.com/l/ludex) pra apoiar.

**Importante:** o Ludex **não distribui ROMs nem BIOS** — não abra issues pedindo isso.

## 🇬🇧 English

Ludex is a **paid** app built in the open — the code is MIT, and the community helps make it better.

- 🐛 **Bug?** Open an [issue](https://github.com/Paulothedeveloper/ludex/issues/new/choose) with: what happened, what you expected, steps to reproduce, your OS (Windows/Android) and Ludex version (Settings → About).
- 💡 **Idea/feature?** Open an issue describing the pain it solves.
- 🌍 **Translation?** The dictionary is in [`src/ludexI18n.js`](src/ludexI18n.js) (6 languages). Fix PRs welcome.
- ⭐ Like it? Star the repo and [buy Ludex](https://pauloadriel98.gumroad.com/l/ludex) to support.

**Note:** Ludex does **not** distribute ROMs or BIOS — please don't request that.
